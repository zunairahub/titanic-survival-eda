# Titanic Survival Analysis — EDA

Exploratory data analysis on the Titanic dataset to identify key survival predictors.

## Key Findings
- Female survival rate: 74% vs Male: 19%
- 1st class survival: 63% vs 3rd class: 24%
- Higher fare strongly correlated with survival
- Children had higher survival rates than adults

## Tools Used
Python · Pandas · Seaborn · Matplotlib · Kaggle
